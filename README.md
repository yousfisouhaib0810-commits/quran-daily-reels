# 🕌 بوت فيديو القرآن اليومي

بوت يُنتج فيديو ريلز يومياً يحتوي على آيات من القرآن الكريم مع النص العثماني والترجمة الإنجليزية.

## ✨ المميزات

- ✅ توليد فيديو يومي تلقائي (9:16 للريلز)
- ✅ 20 قارئ مختلف (اختيار عشوائي)
- ✅ تتابع الآيات من سورة لسورة
- ✅ نص عثماني + ترجمة Sahih International
- ✅ خلفيات متنوعة من Pexels
- ✅ مستشار خلفيات بالذكاء الاصطناعي (OpenRouter)
- ✅ فلترة تلقائية لأي فيديو يحتوي على أشخاص
- ✅ رفع تلقائي إلى Google Drive
- ✅ نشر تلقائي إلى TikTok عبر Buffer
- ✅ يعمل على GitHub Actions (مجاني)

## 🚀 الإعداد

### 1️⃣ Fork المشروع
اضغط على زر "Fork" في أعلى الصفحة لنسخ المشروع لحسابك.

### 2️⃣ الحصول على المفاتيح

#### Pexels API Key (للخلفيات)
1. اذهب إلى: https://www.pexels.com/api/
2. سجّل حساب مجاني
3. انسخ الـ API Key

#### OpenRouter API Key (للاقتراحات الذكية)
1. اذهب إلى: https://openrouter.ai/
2. أنشئ حساباً مجانياً ثم احصل على مفتاح API
3. ضمن "Rate Limits" اختر نموذجاً سريعاً مثل `google/gemini-flash-1.5-8b`
4. انسخ المفتاح لاستخدامه في GitHub Secrets

#### Google Drive API (للرفع)
1. اذهب إلى: https://console.cloud.google.com/
2. أنشئ مشروع جديد
3. فعّل Google Drive API
4. أنشئ OAuth 2.0 Client ID (نوع: Desktop App)
5. حمّل ملف credentials.json

### 3️⃣ إضافة الـ Secrets في GitHub

اذهب إلى: Settings → Secrets and variables → Actions

أضف هذه الـ Secrets:

| الاسم | القيمة |
|-------|--------|
| `PEXELS_API_KEY` | مفتاح Pexels API |
| `OPENROUTER_API_KEY` | مفتاح OpenRouter لاقتراح الخلفيات (اختياري لكن مستحسن) |
| `GOOGLE_CREDENTIALS` | محتوى ملف credentials.json |
| `GOOGLE_DRIVE_FOLDER_ID` | معرف مجلد Google Drive (اختياري) |
| `BUFFER_API_KEY` | مفتاح Buffer API |
| `BUFFER_TIKTOK_ID` | معرف حساب TikTok داخل Buffer |

### 4️⃣ تفعيل Workflow

1. اذهب إلى: Actions
2. اضغط على "Daily Quran Reel Generator"
3. اضغط "Enable workflow"
4. للاختبار: اضغط "Run workflow"

## ⚙️ الإعدادات

يمكنك تعديل `config/config.json`:

- `timing.min_duration_seconds`: الحد الأدنى لمدة الفيديو (15 ثانية)
- `reciters`: قائمة القرّاء
- `pexels.search_queries`: كلمات البحث للخلفيات
- `ai_background`: تفعيل واختيار نموذج الذكاء الاصطناعي وعدد المحاولات لكل فيديو
- `person_filter`: إعدادات فلترة الأشخاص (عدد الإطارات، عتبة الثقة، حجم التصغير)
- `schedule.hour`: ساعة التشغيل اليومي
